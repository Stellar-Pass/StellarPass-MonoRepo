# Stellar Pass Brand Assets

## Logo Variants

| File | Usage |
|---|---|
| `logo-mark.svg` | Icon-only mark — favicons, app icons, social avatars, compact spaces |
| `logo-icon.svg` | Detailed ticket+star icon — hero sections, splash screens |
| `logo-full-dark.svg` | Full logo with wordmark — dark backgrounds |
| `logo-full-light.svg` | Full logo with wordmark — light backgrounds |

## Colors

| Name | Hex | Usage |

| Stellar Purple | `#7B3FE4` | Gradient start, primary accent |

| Dark | `#0F1629` | Dark background wordmark |
| White | `#FFFFFF` | Light background wordmark, icon overlays |

## Design Concept

The logo combines two elements:

- **Ticket shape** — a rounded rectangle with notch cutouts and a dashed perforation line, representing event ticketing
- **Four-pointed star** — the Stellar network's signature motif, centered in the ticket body
- **Gradient** — purple-to-blue gradient across both the ticket and the Stellar identity

The scan lines at the bottom of the detailed icon add a digital/NFT feel, while corner accent dots reinforce the ticket stub aesthetic.

## Usage

- Use `logo-mark.svg` for square spaces (avatars, favicons, app icons)
- Use `logo-full-dark.svg` on dark or colored backgrounds
- Use `logo-full-light.svg` on light or white backgrounds
- Maintain clear space around the logo equal to the width of one star arm
- Do not stretch, rotate, or recolor the logo
